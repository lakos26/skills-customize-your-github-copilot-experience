# 📘 Assignment: Cleaning Messy Data with Pandas

## 🎯 Objective

Clean and prepare a messy dataset using Pandas. You will practice loading CSV data, fixing inconsistent values, handling missing information, and creating a simple summary of the cleaned data.

## 📝 Tasks

### 🛠️ Load and Inspect the Dataset

#### Description

Open the provided CSV file and inspect the data so you can identify issues that need cleaning.

#### Requirements

Completed program should:

- Load the CSV file with Pandas.
- Display the column names and the first few rows.
- Show basic information about missing values and data types.

### 🛠️ Clean the Data

#### Description

Fix the most common data quality issues in the dataset so the values are consistent and usable.

#### Requirements

Completed program should:

- Standardize text values such as names or categories.
- Handle missing values in a reasonable way.
- Convert columns to the correct data types.
- Remove or fix any obvious duplicate or invalid entries.

### 🛠️ Summarize the Cleaned Results

#### Description

Create a short summary of the cleaned dataset to show that the data is ready for analysis.

#### Requirements

Completed program should:

- Print a summary of the cleaned dataset.
- Show at least one useful statistic or grouped result.
- Save the cleaned dataset to a new CSV file.
